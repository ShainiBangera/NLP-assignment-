**NLP Assignment**

This assignment consists of NLP techniques. It includes text preprocessing and feature engineering to analyze and understand the data.

**Steps Covered**

**1. Text Preprocessing:**

Converted text to lowercase
Tokenized sentences into words
Cleaned the text

**2. Feature Engineering:**

Bag of Words (BoW)
TF-IDF
Word2Vec 

**3. Cosine Similarity:**

Calculated similarity between sentences
Helps to understand how similar two sentences are

**4. Text Classification (Supervised Learning):**

Used Logistic Regression
Classified sentences into positive and negative

**5. Clustering (Unsupervised Learning):**

Used K-Means clustering
Grouped similar sentences into clusters

**Conclusion**

This project helped in understanding the complete NLP workflow, including preprocessing, feature extraction, and applying machine learning models on text data.
